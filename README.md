Get the Mac OS versions from the current system!

# Install

```sh
npm install get-osx-version --save
```

# Usage

Output your current Mac OS version!

```javascript
const {macOSVersion} = require('get-osx-version');

console.log('Mac OS version: ' + macOSVersion());
```
